## 环境搭建

下载代码并执行下列命令:

```shell
$ git clone git@192.168.0.102:root/reactexamples.git
$ cd reactexamples
$ npm install
$ npm start
```

在浏览器中运行：http://localhost:9001/   （用户：Admin 密码：123456）


## 单元测试

```shell
$ npm run test
```
